Por Heloísa Martins Vieira - Junho/2022

Repositório contendo os arquivos utilizados, em prática, durante as aulas do projeto "Escola de Desenvolvedores 2022", da Nota Control. O objetivo destas aulas, aplicadas durante 4 dias, foi apresentar uma introdução às linguagens HTML e CSS.

Os estagiários foram expostos aos conceitos básicos de tags, elementos HTML, primeiras classes CSS até a construção de um pequeno projetinho com Bootstrap 5.
